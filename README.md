# n8n API AI Automation

Projeto de automação desenvolvido com n8n, integrando uma API REST própria em Node.js com inteligência artificial local via Ollama.

## Objetivo

Automatizar a análise de dados provenientes de uma API, utilizando workflows para consumir informações, processá-las com IA e gerar respostas automatizadas.

## Tecnologias

- Node.js
- Express
- REST API
- n8n
- Ollama
- Phi-3
- JSON

## Funcionalidades

- Consumo de API REST
- Integração entre serviços
- Processamento automatizado
- Análise de dados com IA
- Workflow de automação
